Задание по кт2

В квадратных скобках указаны баллы за задание
Задание требуется выполнить не меняя html ниже: 
    <body>    
        <div class="parent">
            <div class="card"></div>
            <div class="card"></div>
            <div class="card"></div>
        </div>
        <button id="add">Добавить</button>
        <button id="remove">Удалить</button>
        <button id="change">Изменить</button>
    </body>

C помощью css задать
	

Свойства элемента parent:
	
рамка синего цвета	[1]
margin:40px со всех сторон	[1]
flex контейнер	[2]
Выравнивание элементов внутри него по горизонтальной оси слева-направо	[2]

	

Элемент card, внутри элемента parent:
	
Ширина 100px Высота 250px margin - 20px со всех сторон, рамка черного цвета	[2]
Вид курсора мыши pointer	[1]
У элемента card с классом selected фон оранжевого цвета	[1]
	

При наведении мыши на элемент parent

Внутри всех элементов card появляется кружок диаметром 20px зеленого цвета в левом верхнем углу	[2]
Внутри первого и последнего элемента card появляется кружок диаметром 30px красного цвета в правом нижнем углу	[3]
Появление кружков просходит плавно от точки до круга за 2 секунды.	[3]
У каждого нечетного элемента card кружок при появлении синего цвета и плавно меняет свой на конечный	[3]
	

С помощью js или jQueryl
	
Щелчок по элементу card  переключает у него класс selected	[3]
Проверяется что элемент card имеет непосредственного родителя элемент parent	[3]
Кнопка добавить, добавляет новый элемент card в конец элемента parent	[3]
Кнопка изменить удаляет все элементы card с классом selected	[5]
Кнопка изменить переключает у всех элементов card класс selected	[5]
Дополнительно до 10 баллов (за текущий контроль) можно заработать прислав 2 варианта решения на чистом js и на jQuery
