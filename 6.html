Свойство box-sizing

По умолчанию свойства css width и height задают ширину и высоту содержимого элемента content без учета padding и border.

Поведение можно поменять, задав свойство в значении box-sizing:border-box;

В этом случае width и height указывают размеры с учетом padding и border
