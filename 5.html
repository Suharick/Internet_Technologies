CSS специфичность (приоритеты правил)

Подсчитывается три типа параметров:
  • A - #id
  • B - классы и псевдоклассы
  • C - теги элементов, например, div или ::before

Записывается десятичное число из цифр ABC.

Формально число не в 10-тичной системе, а в системе с достатосно большим основанием AN^3 + BN^2 + CN, где N достаточно велико.

Чем число больше, тем правило специфичней (приоритетней).

Дополнительно перекрытие !important (не рекомендуется к использованию, так как перекрываются все css правила). Имеет наивысшую приоритетность.

Свойства, указанные в html через style элемента важнее других правил css (но не приоритетней !important) и тоже не рекомендуется для использования
