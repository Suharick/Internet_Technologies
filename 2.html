CSS - Cascading Style Sheet

CSS может быть в отдельном файле обычно с расширением css, так и внутри html в теге style

Универсальные свойства HTML элементов:

  • id - уникальный идентификатор, на странице повторяться не должен
  • class - может повторяться. Может быть много классов у одного элемента. Записываются через пробел.

Некоторые CSS свойства

  • color - цвет шрифта, например, red
  • background - цвет фона
  • font-size - размер шрифта, например, 20px

CSS Selector

Примеры:
  • tag, например, "p"
  • #id
  • .class

Блочная модель

Свойства CSS: padding, margin, border.

Разница padding и margin:
  • padding внутри рамки border, margin - вне;
  • padding имеет цвет фона, margin всегда прозрачный;
  • вертикальные margin у соседних элементов перекрываются (схлопываются);
  • также схлопываются margin у родителя и детеныша;
  • margin допустим отрицательным, padding не допустим быть отрицательным.
Запись свойств padding и margin:
  • 1 число, например, 20px (одинаковые со всех сторон)
  • 2 числа, например, 20px 10px (первое - верхняя и нижняя, второе - левая и правая)
  • 4 числа, например, 20px 10px 5px 1px (top, right, bottom, left)
